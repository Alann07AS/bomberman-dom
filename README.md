# bomberman-dom

Multiplayer bomberman game with local and only(LAN) player for now you can play with ZQSD/WASD, Arrows and PS4 Gamepad (digital arrows and X).

[LinkToGame](https://alann07as.github.io/bomberman-dom/client/)
